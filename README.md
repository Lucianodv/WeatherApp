# WeatherApp

Esta app de clima obtiene nuestra ubicacion y ademas podemos seleccionar una ciudad para obtener su informacion climatica.

Utiliza: Retrofit, Gson, Recyclerview, Interceptors, okHttp3, API WEATHER.
